# Sensores-Java

Este repositorio contiene una práctica desarrollada en **Java** para trabajar con sensores. La implementación forma parte de los ejercicios realizados en clase para aplicar programación orientada a objetos y entrada/salida de datos en Java.

## Descripción

El proyecto incluye código en Java que simula o gestiona datos de sensores, aplicando conceptos de programación aprendidos en la asignatura. La práctica se centra en la lectura y procesamiento de información desde dispositivos (o simulaciones) y su manejo dentro de una aplicación Java.

## Tecnologías utilizadas

- Java
- IDE compatible con Java (por ejemplo, Eclipse, IntelliJ IDEA o NetBeans)
